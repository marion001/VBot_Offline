Bước 1: Sửa sources.list:

	- Chạy lệnh sau:
		$:> sudo nano /etc/apt/sources.list

	- Chú thích dòng này (Thêm Dấu # vào Trước)
		#deb http://raspbian.raspberrypi.org/raspbian/ bullseye main contrib non-free rpi

	- Sau đó thêm mới dòng này:
		deb http://archive.raspbian.org/raspbian bullseye main contrib non-free
		- Nhấn Ctrl + X => nhấn phím chữ "y" => nhấn phím "Enter"

	- Chạy 2 lệnh sau để áp dụng:
		$:> sudo apt clean
		$:> sudo apt update

Bước 3: Cài Rpi-SetWiFi-viaBluetooth
	- Chạy lệnh sau:
		$:> curl  -L https://raw.githubusercontent.com/nksan/Rpi-SetWiFi-viaBluetooth/main/btwifisetInstall.sh | bash
		
	- Khi được hỏi: Specify btwifiset service install location
		sẽ nhấn enter để cài thư mục mặc định

	- Khi được hỏi: Bluetooth password (encryption key) [Default: VBot-Assistant]:
		sẽ nhập: vbot123

	- Khi được hỏi: btwifiset needs your WiFi Country code
		sẽ nhập: vn

	- Nhập Xong Chương trình sẽ bắt đầu cài Rpi-SetWiFi-viaBluetooth
	- Khi cài thành công nhập lệnh sau để khởi động lại hệ thống:
		$:> sudo reboot
